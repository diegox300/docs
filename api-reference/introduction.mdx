# API de Atualização de Veículos

Bem-vindo à documentação da API interna para atualização de status de veículos. Esta API é projetada para receber dados de um sistema externo, validar a requisição e atualizar as informações correspondentes em nosso banco de dados Supabase.

<Info>
  Esta API é uma **Edge Function** executada diretamente na infraestrutura do
  Supabase, garantindo baixa latência e integração direta com o banco de dados.
</Info>

### Principais Funcionalidades

- **Autenticação Segura**: Utiliza um token `Bearer` para proteger o endpoint.
- **Atualização Atômica**: Atualiza o status de um veículo e registra o payload e a data da operação.
- **Validação de Dados**: Garante que os dados recebidos estejam no formato correto.
- **Respostas Claras**: Fornece feedback claro sobre o sucesso ou a falha de cada requisição.
